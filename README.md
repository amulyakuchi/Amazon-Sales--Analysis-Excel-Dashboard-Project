# Amazon-Sales--Analysis-Excel-Dashboard-Project


A fully interactive **Excel Sales Analysis Dashboard** built using real-world Amazon sales data.
This project demonstrates skills in **data cleaning, preprocessing, PivotTables,  and dashboard design** — delivering clear business insights for decision-making.


 Key Highlights

* Cleaned and transformed raw dataset (removed duplicates, standardized dates, fixed missing values).
* Created new calculated fields:

  * **Revenue**,
  *  **Total Cost**,
  *   **Month**,
  *    **Category**
* Built multiple PivotTables for:

  * Category-wise revenue
  * Top 10 products
  * Monthly trends
  * City-wise performance
  * Payment method distribution
  

Interactive Dashboard Features

* Revenue by Category (Column Chart)
* Top 10 Products (Bar Chart)
* Monthly Sales Trend (Line Chart)
* City-Wise Sales (Map/Bar)
* Payment Mode Share (Doughnut)
**Slicers:**
**Category,
*** City,**
**Month**
**Delivery Status**

Tools & Skills Used

* Microsoft Excel
* Data Cleaning & Preprocessing
* PivotTables
* Data Visualization
* Dashboard Design

What’s Included

* **amazon_sales_data 2025.xlsx** — cleaned dataset + dashboard
* PivotTables, charts, slicers, in a single workbook

Outcome

A polished analytical dashboard that converts raw sales data into **actionable business insights** — showcasing strong Excel, analytics, and reporting skills.
